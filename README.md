## ready

> dotfiles for linux and mac

### mac

```sh
`xcode-select --install`

# then
./other/install-common.sh
./sync.sh
```

optionally, there are installation steps in `./ohter/macports.sh`

### linux

```sh
./other/install-common.sh
./sync.sh
````

optionally, there are installation steps in `./ohter/install-ubuntu.sh`
